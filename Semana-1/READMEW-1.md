# Semana 1 - Ejercicios de Java

Esta carpeta contiene los ejercicios realizados durante la primera semana de práctica con Java. Los proyectos están organizados en carpetas independientes dentro del directorio `semana-1`, cada uno enfocado en diferentes conceptos básicos y orientados a reforzar la lógica de programación y la programación orientada a objetos.

## Estructura

```
semana-1/
├── 1 HolaMundoMejorado/
├── 2 Calculadora Basica/
├── 3 Clasificador de Nmeros/
├── 4 Manipulador de Strings/
├── 5 Gestion de Estudiantes/
└── 6 Herencia de Vehiculos/
```

## Proyectos

### 1. Hola Mundo Mejorado

Programa sencillo donde se trabajan variables primitivas, cadenas de texto y salida por consola. Se genera un mensaje de presentación utilizando concatenación y también mediante `String.format()` para conocer distintas formas de construir textos.

Conceptos utilizados:

* Variables primitivas
* String
* Concatenación
* Formato de cadenas

---

### 2. Calculadora Básica

Aplicación que implementa operaciones de suma utilizando sobrecarga de métodos. Dependiendo de los parámetros recibidos, Java selecciona automáticamente el método adecuado.

Conceptos utilizados:

* Sobrecarga de métodos
* Métodos estáticos
* Arreglos
* Tipos de datos numéricos

---

### 3. Clasificador de Números

Proyecto encargado de recorrer una colección de números y clasificarlos según distintas características. Además calcula estadísticas simples como promedio, valor máximo y mínimo.

Conceptos utilizados:

* Ciclos `for`
* Condicionales `if/else`
* Operadores lógicos
* Arreglos
* Estadísticas básicas

---

### 4. Manipulador de Strings

Conjunto de utilidades para trabajar con texto. Incluye funciones para invertir cadenas, detectar palíndromos, contar vocales y generar una pirámide utilizando caracteres.

Conceptos utilizados:

* StringBuilder
* Métodos de String
* Manipulación de caracteres
* Bucles

---

### 5. Gestión de Estudiantes

Sistema básico para administrar estudiantes y sus calificaciones. Se pueden registrar notas, calcular promedios y determinar qué estudiante obtuvo el mejor resultado.

Conceptos utilizados:

* Encapsulación
* Getters y setters
* ArrayList
* Métodos personalizados
* Sobrescritura de `toString()`


---

### 6. Herencia de Vehículos

Proyecto orientado a practicar herencia, clases abstractas, interfaces y polimorfismo. Se modela una pequeña flota de vehículos formada por autos, motocicletas y vehículos eléctricos.

Conceptos utilizados:

* Herencia
* Polimorfismo
* Interfaces
* Clases abstractas
* Casting implícito mediante referencias padre

---

## Objetivo General

El propósito de estos ejercicios fue reforzar los fundamentos de Java antes de avanzar hacia temas más complejos que se verán a lo largo de esta academia.

Estos proyectos fueron desarrollados con fines académicos y de práctica para mejorar la comprensión de la POO asi como la lógica de programación.
