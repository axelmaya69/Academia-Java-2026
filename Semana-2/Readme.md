# Ejercicios Java

En esta semana se se realizaron una serie de ejercicios enfocados en características avanzadas de Java como excepciones 
personalizadas, clases anidadas, enums avanzados, interfaces funcionales y patrones de diseño.

## Ejercicio 1: Sistema Bancario con Excepciones Personalizadas

Simula operaciones bancarias como depósitos, retiros y transferencias entre cuentas.

Conceptos practicados:

* Excepciones checked y unchecked
* Excepciones personalizadas
* Try-with-resources
* AutoCloseable
* Multi-catch
* Validación de reglas de negocio

---

## Ejercicio 2: SimpleStack con Inner Class

Implementa una estructura de datos tipo pila (Stack) genérica utilizando nodos enlazados.

Conceptos practicados:

* Clases anidadas estáticas (Static Nested Classes)
* Inner Classes
* Generics
* Iterable e Iterator
* Uso de for-each personalizado
* Manejo de excepciones en estructuras de datos

---

## Ejercicio 3: Sistema de Tickets con Enums Avanzados

Gestiona tickets de soporte con prioridades y estados, incluyendo reglas de transición entre estados.

Conceptos practicados:

* Enums con atributos y constructores
* Métodos dentro de enums
* EnumMap
* EnumSet
* Modelado de estados
* Validación de transiciones

---

## Ejercicio 4: Product Catalog Pipeline

Procesa un catálogo de productos mediante filtros, transformaciones y acciones encadenables.

Conceptos practicados:

* Interfaces funcionales
* Predicate
* Function
* Consumer
* Supplier
* Method References
* Programación funcional en Java

---

## Ejercicio 5: Pizza Builder + Decorator

Simula un sistema de pedidos de pizza donde se construyen pizzas personalizadas y se agregan extras que modifican su precio.

Conceptos practicados:

* Patrón Builder
* Patrón Decorator
* Enums con comportamiento
* Validación durante la construcción de objetos
* Composición de funcionalidades

---

## Ejercicio 6: Event Framework

Implementa un sistema de eventos para órdenes utilizando publicación de eventos y estrategias de notificación intercambiables.

Conceptos practicados:

* Patrón Observer
* Patrón Strategy
* Inyección de Dependencias (Dependency Injection)
* Interfaces funcionales
* Lambdas
* Excepciones checked
* Arquitectura desacoplada basada en eventos

---

## Objetivo General

Estos ejercicios sirven como práctica integral de Java moderno y cubren los teams del temario de la academia
para entender más sobre Java y la programacion orientada a objetos.
